# Update SUAS 1.0.2

Conteúdo publicado para o repositório público conectcras-ai/Update-SUAS.

Arquivos principais:
- manifest.xml: app.version 1.0.2.
- app/cras-gestao-suas-app-1.0.2-all.jar.

Correção:
- Recursos internos do SUAS agora usam o novo ícone do desktop.
- Versão 1.0.2 mantém a correção de alerta de visitas domiciliares da 1.0.1.

Observação:
- O update por JAR atualiza a aplicação. Para corrigir o ícone do atalho do Windows, use o instalador ConectSUAS-1.0.2.exe.
